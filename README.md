# Greed & Fear Dashboard

Greed & Fear is a Aave user position dashboard built to provide an intuitive overview of user asset allocations, key metrics, and protocol data. It calculates Greed and Fear index of the protocol.

## Technologies Used

- NextJS
- GraphQL: Fetches real-time data from a subgraph for protocol metrics.
- EthersJS
- Aave utilities

## Getting Started

### Installation

Clone the repository:

```bash
git clone https://github.com/auralshin/aave-craft-task.git
cd aave-craft-task
```

Install dependencies:

```bash

yarn install
```

Start the development server:

```bash
yarn dev
```

Open your browser at ```http://localhost:3000```
